How this site is built

This is a static site, generated from a bunch of Markdown. The generator used is the awesome Pandoc. To orchestrate the build process, some Bash and Ruby is thrown in the mix.

I wanted a very simple way to generate my static site. Using things like Hexo or Jekyll seemed such an overkill. So I pieced some Bash and Ruby together. The end result is beautiful. You can see the build script for yourself :)

Want to suggest edits or changes to this site’s content? Please submit a pull request. This site’s source code is available publicly.
