How this site is built

This is a static site, generated from a bunch of Markdown. The generator used is the awesome Pandoc. Some Bash and Ruby is used to orchestrate the build process.

I wanted a very simple way to generate my static site, and using things like Hexo or Jekyll seemed like an overkill to me. So I pieced some Bash and Ruby together, and it works beautifully. You can see the code for yourself :)

Want to suggest edits or changes to this site’s content? Please submit a pull request - this site’s source code is available publicly.
